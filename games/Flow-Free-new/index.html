How to Play Flow Free

Connect matching colors with pipes to create a flow. Fill the entire board to solve each puzzle!

Rules:
  1. Connect dots of the same color with a continuous pipe
  2. Pipes cannot cross or overlap each other
  3. Every cell on the board must be filled with a pipe
  4. You can change your path at any time

Guaranteed Solvable Puzzles:

Every puzzle generated is guaranteed to have at least one solution. The game creates a complete solution first, then removes the paths to create the puzzle.

Tips:
  • Start with colors that have limited options
  • Look for dots that are already close to each other
  • Use the edges and corners strategically
  • If stuck, use a hint to reveal part of a solution
